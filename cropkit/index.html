CROP·KIT — Help
Workflow
Upload
Click + or drag images in. Supports JPG, PNG, WebP, GIF, AVIF, BMP, SVG, and HEIC (converts automatically). File sizes shown in sidebar.
Preset
Select a size above the canvas, or enter custom W × H. Crop frame scales to fill available space.
Drag / scroll
Drag to pan. Scroll to zoom. Frame turns green when fully inside image bounds. Edges snap to align.
Crop →
Exports at full resolution. Auto-advances. Output size shown on each card.
All Sizes
Crops current position across all four standard presets at once.
Copy pos
Toggle (C) to carry zoom/rotation/flip/pan into the next image instead of resetting.
Focal memory
Clicking a previous image in the sidebar restores your last position for it. Teal dot = focal point saved.
Keyboard Shortcuts
Enter
Crop at current preset.
Space
Skip and advance.
R
Reset transform.
↑↓←→
Nudge 1px. Hold Shift for 10px.
A
Crop all presets.
C
Toggle copy position to next.
Ctrl Z
Undo last transform (up to 30 steps).
?
Open this panel.
Esc
Close dialogs.
Canvas Overlays
Grid
Cycles: off → rule of thirds → golden ratio (38.2%) → 4×4 grid. A darkening overlay appears inside the frame to improve line contrast.
Safe zone
Dashed inset guide calibrated per preset. Instagram 14%, Hero 20%/15%, Card 10%, Thumbnail 20%. Also triggers the darkening overlay.
Output Options
Quality
Controls lossy compression for WebP/JPEG (50–100). Grayed out for lossless PNG.
Fill color
Canvas background when image doesn't cover the frame. "none" = transparent (best with PNG).
Filename Tokens
{name}
Original filename without extension.
{preset}
Preset ID: ig, hero, card, thumb, or custom.
{dims}
Output dimensions, e.g. 1080x1080.
{date}
Today's date: YYYY-MM-DD.
Auto sep
Clicking tokens back-to-back inserts _ separator automatically.
CROP·KIT — Changelog
v1.6
Current
  • Font size +20% — Base increased from 16px to 19px across all UI text, labels, buttons, modals, and controls.
  • Darkening frame overlay — When the grid or safe zone is toggled on, a semi-transparent dark layer renders inside the crop frame, making guide lines and the dashed safe zone border clearly visible against any image content.
  • Grid visibility — Grid lines increased to 55% white opacity with stronger anti-alias width. Safe zone border increased to 2.5px solid yellow with a drop shadow for contrast.
  • HEIC/HEIF import — iPhone photos load via heic2any library. The image shows "converting…" in the sidebar while processing, then becomes available in the editor as a standard PNG.
  • AVIF + BMP import — Added to accepted formats. Both decode natively in modern browsers, no library required.
  • SVG import — SVG files load via canvas drawImage and render at full vector quality. Note: SVGs without explicit width/height attributes may need a viewBox to size correctly.
